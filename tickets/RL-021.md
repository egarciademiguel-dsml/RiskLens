# RL-021 — VaR Backtesting

## Status: Done

## Goal
Validate VaR predictions with rolling-window backtesting and statistical tests (Kupiec, Christoffersen).

## What changed
- **New module `src/analytics/backtesting.py`**: `backtest_var()` rolling-window engine, `kupiec_test()` unconditional coverage, `christoffersen_test()` independence, `backtest_summary()`, predefined fit functions for all 5 model types
- **Tests**: 23 tests covering Kupiec edge cases, Christoffersen independence/clustering, backtest integration with constant and GARCH, summary validation
- **Streamlit**: New "VaR Backtest" tab with model selector, window/step sliders, breach chart with pass/fail badges
- **Notebook**: New section 12 in `model_deep_dive.ipynb` — backtests Constant, GARCH, HMM-2, XGB-21d with summary table and per-model breach charts
- **Math reference**: Added HMM, GMM, XGBoost theory sections. Added backtesting theory (Kupiec LR, Christoffersen independence, joint test). Updated model combinations table and assumptions.
